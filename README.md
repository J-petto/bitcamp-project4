## 주요 기능
- ServerApp으로 서버 구축<br><br>

- 2명의 클라이언트가 빙고 게임 플레이
    1. 나만의 닉네임 정하기.
    2. 1~25까지 랜덤한 숫자 빙고판 자동 생성.
    3. 빙고판 숫자입력으로 빙고 채우기.
    4. 먼저 3빙고를 완성하면 승리.
    5. 게임 종료.
